## Prerequisite

You need Node.JS `v16+` to run this application

```sh
$ node -v
v16.11.1
```

## Installation

Install npm dependencies

```sh
npm install
```

Copy `.env.dev` file into `.env` file and fill it

```sh
cp .env.dev .env
```

## Usage

Start the application

```sh
npm start
```

Start the application in development mode (with hot reloading)

```sh
npm run dev
```
